## Project setup

### Install deps

```sh
# yarn
yarn

# npm
npm install

# pnpm
pnpm install

# bun
bun install
```

### Compiles and hot-reloads for development

```sh
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev

# bun
bun run dev
```

### Compiles and minifies for production

```sh
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build

# bun
bun run build
```
